# Detray Tutorial

### Prerequistes
1. GCC >= 10.3.1
2. CUDA >= 11.5 and any GPU device

### Clone the repository and Compile

```sh
git clone https://github.com/beomki-yeo/detray_tutorial.git
cd detray_tutorial
mkdir build
cd build/
cmake ../
make
```

### Run propagator example

```sh
./bin/detray_tutorial_propagator
```
